# Project Echo Client

**Echo Client** is a modded Minecraft 1.8.8 client focused on maximizing performance and maintaining a lightweight footprint, while offering a clean and responsive user interface.  
It supports **Windows**, **Linux**, and **macOS**.

---

## Features

> ### Auto GG  
>
> ### Bedwars HUD  
> - Arrows  
> - Resources  
>
> ### Custom Chat  
>
> ### Custom Hotbar  
>
> ### Game Joiner  
> - Murder Mystery  
> - Skyblock  
> - Solo/Doubles/3s/4s Bedwars  
> - Solo/Team Normal/Insane Skywars  
> - The Pit  
>
> ### Info HUD  
> - Biome  
> - Coordinates  
> - CPS  
> - Direction  
> - FPS  
>
> ### Item Physics  
>
> ### Keystrokes  
>
> ### Potion HUD  
>
> ### Toggle Sprint  

---

## Built For
- Minecraft **1.8.8**
